# Shop
Project for "Database Systems". 
Project was created in hurry so it's not the best quality.
Used technologies:
- MVC .NET 4.5
- Entity Framework 6.0 used with code first approach
- some basic jquery
Functionalities:
- Admin:
  - can manage assortment
    - product categories
    - manufacturers
    - products
- User:
  - can browse products
  - manage cart and place order
